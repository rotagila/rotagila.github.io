Cataclimax 2119

Unity - Arcade - 2D

Cataclimax 2119 was produced in 48 hours for Wonderjam 2019, which is a game jam organized by UQAC (Université du Québec À Chicoutimi). The theme was "Climate disaster in 2119: when the end of the world is near!". 3 styles were also imposed on our team: "Arcade", "Management", and "Adventure". We chose to go with an "Arcade/Management" idea. Another restriction was that the game was to be played on an arcade machine, which was interesting because we had to think about the game design differently compared to a "classic" game.
Here's a short description of the game : in 2019, many Disasters appear in the world. The goal is to stop them before they actually happen. The difficulty gradually increases as time flows : the player must manage his Counter-Disaster charges and choose to abandon some parts of the world in favour of others. The game is over when a region is completely doomed. Starting from the year 2119, the game reaches its peak of difficulty, which means that A LOT of Disasters are coming!
The player's choices can also be influenced by each region's weaknesses and resistances to each type of Disaster. These must be discovered and used by the player to reach the highest scores!
In this project, my main responsibilities were :

  • Counter-Disaster system (core mechanic)
  • HUD
  • Controls
  • Damage system
  • Time flow
  • Help screen
  • I also helped with menus, balancing and overall support to the team.

Team: Dorian ANTOINET, Victor DELENCLOS, Juliette LALMAN, Clément RAMAT
Completed in : October 2019

Orbs

Unity - Puzzle - 2D

Orbs was produced in 24 hours for Extra Jam 3, which is a game jam organized by Extra Life. The theme was "15 items or less".
Orbs is a simple and colourful looking game with 10 playable levels. Each level has a common objective : to retrieve all the colored orbs with a minimized number of moves. The player can manipulate his own orb with the mouse by adjusting the angle as well as the shooting power. On its way, the orb may encounter gadgets that will make it accelerate, bounce, attracted or even teleport him somewhere else! With each orb absorbed, the player's orb will grow (up to a certain limit) and will be able to aim for larger orbs.
Orbs makes extensive use of the particle system and Unity physics.
In this project, my main responsibilities were :

  • In-game visuals
  • HUD
  • Particle effects
  • Assets creation (all but sounds)
  • Acceleration and Rebound gadgets (core mechanics)
  • I also helped with level design (especially levels 3, 8, 9 and 10) and overall support to the team.

Team: Dorian ANTOINET, Victor DELENCLOS, Augustin LEO, Clément RAMAT
Completed in : November 2019

0-GUST

Unity - Stealth - 2D

0-GUST is the result of my very first game jam : the 45th edition of the Ludum Dare. The duration was 72 hours, and the theme was "Start with nothing". My team chose to make a stealth game.
The goal is, as a small microwave oven named 0-GUST, to escape a disassembly plant. At first, you don't have any ability and you should get caught pretty quickly. So you have to manage to unlock different abilities as you progress through the game!
As this was our first game jam, we made a significant number of errors related to time management and task allocation/prioritization. This generated a number of bugs that we were not able to fix in time. After the jam was over, we fixed most of the bugs. Furthermore, we chose to leave only one of the abilities we had created in the game in order to simplify the gameplay (we went a bit overboard and created things like an ice-thrower, an EMP or even a flamethrower...). These could have been interesting, but the code architecture as a whole wasn't really adapted to handle them anyway.
Precisely because we have made so many mistakes, this experience is valuable.
In this project, my main responsibilities were :

  • Ability system (core mechanic): design of the abilities, unlocking, usage
  • Dialog system (secondary mechanic)
  • I also provided overall support to the team

Team: Dorian ANTOINET, Victor DELENCLOS, Clément RAMAT
Completed in : October 2019

Network platformer

Unreal Engine - Network - 3D

This was produced as part of a school project. The goal was to improve as Developers on Unreal Engine, and also to learn how to do Network Programming on a game engine used in the industry.
The style "Platformer" was imposed. We created two levels. The first level was used to "setup" the project with Chaos, Niagara and some gameplay programming. The second level's goal was really to use photo-realistic assets and to implement another gameplay mechanic.
In this project, other than Network Programming, you will find:

  • Chaos Destruction (for this, we created and used a custom build derived from Unreal Engine 4.23)
  • Niagara particle systems
  • Shaders
  • UMG UIs
  • Photo-realistic assets (Paragon's Shinbi, Quixel)

Team: Dorian ANTOINET, Victor DELENCLOS, Raphaëlle GARCIA
Completed in : December 2019
